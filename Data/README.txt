README 

When the download of the original dataset is done, run the dataprocess code, it will gives you the two set of dataset.
The array named data_xxxx.csv comprises the datasets that only have the variables comprised in the [0,1] interval.
The array named DATA_xxxx_fin.csv comprises the final datasets with a full pre-processing

The histograms of the pre-processed variables are visibles in the folder DATA_figures
