README 

When the download of the original dataset is done, run the dataprocess code, it will gives you the two set of dataset.
The array named data_xxxx.csv comprises the datasets that only have the variables comprised in the [0,1] interval.
The array named DATA_xxxx_fin.csv comprises the final datasets with a full pre-processing

The Figure_1,3,5,7,9,11 are the plot of the variables histograms of the signal only and signal + background with only the constraint of being in the [0,1] interval with the cumulative function of the signal for the given variable beside.
The Figure_2,4,6,8,10,12 are the histograms of the vaariables after the whole pre-processing
